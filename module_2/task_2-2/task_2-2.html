1. 1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Constants can be used for values that will remain unchanged. They can help with keeping your code tidy, as you only need to change the value in one place, instead of going through yout whole code and change several places. It is also easier for understanding the code if you give your values names, so you know what they are used for instead of having to remember every values task. Variables can be used for data that possibly can change throughout you coding, such as user input, changes or temporary values that is going to be updated. If you use data that can vary, for example database input, you will need to use variables. They can not all be constants, because this can cause errors in your program, and to create a dynamic, useable code you will need to have variables.
